# css-type-base
[![NPM version][npm-image]][npm-url]
[![Downloads][downloads-image]][downloads-url]

Scalar CSS typography. Extracted from [typebase][0]

## Installation
```sh
$ npm install css-type-base
```

## Usage
With [sheetify][2] installed do:
```css
@import 'css-type-base';
```
